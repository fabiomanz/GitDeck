# GitDeck configuration

**repo_path**
Absolute path to the folder inside your git repository that contains the CrowdAnki deck files.
Example: `/Users/you/Decks/my-anki-repo`

**deck_name**
Exact deck name as it appears in Anki. Use `::` separators for sub-decks.
Example: `Japanese::Core 2000`

---

Both fields must be set before using Pull or Push.
